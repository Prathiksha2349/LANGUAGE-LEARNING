Lesson 1: Hello, World!

Let's start with a simple program. Type the following code into the editor:

#include <stdio.h>
int main() {
    printf("Hello, World!\n");
    return 0;
}

Lesson 2: Variables and Input

Now, let's learn how to use variables and take input from the user. Type the following code:

#include <stdio.h>
int main() {
    int age;
    printf("Enter your age: ");
    scanf("%d", &age);
    printf("Your age is %d\n", age);
    return 0;
}
